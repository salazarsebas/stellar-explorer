---
title: Exploring Accounts
description: How to look up and understand Stellar accounts.
---

Every participant on the Stellar network has an account identified by a public key (starting with `G`).

## Looking Up an Account

Paste any public key into the search bar, or click an account address anywhere in the explorer.

## Account Details

The account page displays:

- **Balances** — XLM and all trusted assets with their amounts
- **Trustlines** — Which assets the account can hold
- **Signers** — Keys authorized to sign transactions for this account
- **Thresholds** — Security levels for different operation types
- **Data Entries** — Key-value data attached to the account

## Account Activity

Below the account details, three tabs show the account's history:

- **Transactions** — All transactions involving this account
- **Operations** — Individual operations affecting this account
- **Effects** — Granular changes to the account's state

Results are paginated. Scroll down to load more.
