---
title: Cambiar de red
description: Cómo cambiar entre Public, Testnet y Futurenet.
---

Stellar Explorer soporta tres redes. La red actual se muestra en el encabezado y se refleja en la URL.

## Redes disponibles

| Red | Propósito |
|---|---|
| **Public** | La red Stellar de producción con activos reales |
| **Testnet** | Un sandbox para desarrollo y pruebas |
| **Futurenet** | Una red experimental para funciones futuras |

## Cómo cambiar

Haz clic en el selector de red en el encabezado para cambiar. La página se recargará con datos de la red seleccionada.

## Estructura de URL

La red está incorporada en la ruta de la URL:

```
/es/public/transactions    → Red Public
/es/testnet/transactions   → Testnet
/es/futurenet/transactions → Futurenet
```

Los marcadores y los enlaces compartidos siempre apuntan a una red específica.
