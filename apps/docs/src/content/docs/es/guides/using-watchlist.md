---
title: Usar la lista de seguimiento
description: Cómo guardar y rastrear cuentas y activos.
---

La lista de seguimiento te permite guardar cuentas y activos que quieras monitorear. Los datos se almacenan localmente en tu navegador.

## Añadir a la lista de seguimiento

En cualquier página de cuenta o activo, haz clic en el ícono de marcador para añadirlo a tu lista de seguimiento.

## Ver tu lista de seguimiento

Navega a la página de Lista de seguimiento desde la barra lateral. Los elementos guardados están organizados por tipo:

- **Cuentas** — Claves públicas guardadas con su saldo actual de XLM
- **Activos** — Activos guardados con su código y emisor

## Gestionar elementos

Haz clic en el ícono de marcador nuevamente para eliminar un elemento. Tu lista de seguimiento se almacena en el almacenamiento local del navegador y persiste entre sesiones.

:::tip
La lista de seguimiento es por navegador. Cambiar de navegador o borrar los datos del navegador la restablecerá.
:::
